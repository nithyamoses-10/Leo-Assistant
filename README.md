# AI Virtual Assistant - LEO

## Description
LEO is a Python-based voice-controlled virtual assistant that performs tasks like playing YouTube songs, answering questions, telling time, and fetching information.

## Features
- Voice recognition
- Play YouTube songs
- Google search
- Time and info queries

## Tech Stack
- Python
- SpeechRecognition
- pyttsx3
- pywhatkit

## How to Run
pip install -r requirements.txt  
python project.py
